üìñ Basic Usage Example

Here's how simple it is to use WebSharp in HTML:

<!-- Include WebSharp Runtime --> <script src="websharp-runtime.js"></script> <!-- Write WebSharp code directly in HTML --> <wscript> public class HelloWorld { public static void Main() { var message = new HTMLDivElement(); message.TextContent = "Hello from WebSharp!"; message.Style.Color = "blue"; message.Style.FontSize = "18px"; Document.Body.AppendChild(message); } } HelloWorld.Main(); </wscript>
