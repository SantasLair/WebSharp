The Vision: C#-Style Browser Objects

Imagine writing WebSharp code that feels exactly like working with .NET objects, but for web development. Instead of the lower-level JS.Call() approach, you'd have native C#-style objects like Document, HTMLElement, HTMLButtonElement, etc.

üìä Current vs. Proposed Approach

üîß Current Phase 4: JS.Call()

Lower-level JavaScript interop

var button = JS.Call("document.createElement", "button"); JS.Set(button, "textContent", "Click me!"); JS.Set(button, "style.backgroundColor", "#blue"); JS.Call("document.body.appendChild", button);

‚úÖ Works, but feels verbose and low-level

‚ú® Proposed Phase 5: Native DOM

C#-style object manipulation

HTMLButtonElement button = new HTMLButtonElement(); button.TextContent = "Click me!"; button.Style.BackgroundColor = "#blue"; Document.Body.AppendChild(button);

üéØ Natural, familiar, and type-safe!

üèóÔ∏è Implementation Architecture

Core DOM Classes

  • Document: Static class with CreateElement(), QuerySelector(), etc.
  • Element: Base class with common properties (Id, ClassName, Style)
  • HTMLElement: Enhanced base with web-specific properties
  • Specific Elements: HTMLButtonElement, HTMLInputElement, HTMLDivElement, etc.
  • CSSStyleDeclaration: Type-safe CSS property access
  • Events: MouseEvent, KeyboardEvent with C# event handlers

// WebSharp DOM Runtime Bridge const WebSharpDOM = { Document: { CreateElement: function(tagName, type) { const element = document.createElement(tagName); return new WebSharpElement(element, type || 'Element'); }, get Body() { return new WebSharpElement(document.body, 'HTMLElement'); } }, WebSharpElement: class { constructor(domElement, type) { this._dom = domElement; this._type = type; } get TextContent() { return this._dom.textContent; } set TextContent(value) { this._dom.textContent = value; } get Style() { return new WebSharpCSSStyleDeclaration(this._dom.style); } AppendChild(child) { this._dom.appendChild(child._dom); } } }; // Element constructors class HTMLButtonElement extends WebSharpDOM.WebSharpElement { constructor() { super(document.createElement('button'), 'HTMLButtonElement'); } }
